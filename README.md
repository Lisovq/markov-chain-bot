## Test Chat-bot

This chat-bot example generate messages with mc.api library.
Contains Longpoll/Callback running mode.
Constructed on [vkbottle](https://github.com/timoniq/vkbottle) framework.

### Used libraries
```
[*] aiohttp
[*] ujson
[*] mc.api
[*] peewee
[*] peewee-async
```
